## 1. Что это
**PUMP** — фитнес-приложение «онлайн-тренер» для новичков в зале (рынок СНГ).
Формат: **Telegram Mini App** (PWA на React). Уклон — **в тренировки и онлайн-тренера**; подсчёт калорий вторичен.

- **Зачем Telegram, а не сторы:** нет комиссий 15–30%, нет риска отказа Apple (правило 4.2), быстрый запуск и шеринг. Основной канал продвижения и монетизации.
- **Монетизация (план):** бесплатное ядро + платная подписка (расширенная аналитика, ИИ-фишки, калории по фото — позже, нужен бэкенд).
- **Отличие от конкурентов:** связка «тренировки (план + пошаговый гид + авто-прогрессия) + питание» в одном, под новичка и русский язык. Конкурент по калориям @vkusai_bot — только фото-калории, ранняя стадия.
- **Ориентиры:** Hevy (логирование-first, соц-лента, авто-вес), Fitbod (ИИ-план, карта восстановления), Strong (минимализм), Cal AI / @vkusai_bot (калории по фото).

## 2. Стиль и тематика
- **Тёмная спорт-неон тема**, премиальный вид. Направление — **Frosted Glass** (выбрано пользователем).
- Цвета (CSS-переменные в `styles.css`):
  - фон `--bg #0b0d10`, карточки `--card #161a22`/`--card2 #1d222c`, линии `--line #262c38`
  - акцент `--accent #ff5a1f`, `--accent2 #ff8a3d`, подложка `--accent-soft rgba(255,90,31,.14)`
  - текст `--text #f1f4f9`, приглушённый `--muted #98a2b3`
  - стекло `--glass rgba(19,23,31,.55)` + `backdrop-filter: blur` + окантовка `--glassbd`
- Цвета групп: грудь `#ff5a1f`, спина `#3b82f6`, ноги `#22c55e`, плечи `#a855f7`, бицепс `#ec4899`, трицепс `#f59e0b`, пресс `#14b8a6`.
- **Шрифты:** `Rubik` — текст, `Unbounded` — заголовки и крупные цифры.
- **Тело/мышцы:** фотореалистичные мышечные кропы (PNG, обработаны из присланного рендера; неон приглушён). Карта тела крупная (`muscle_front/back.png`) + 7 зум-кропов групп (`g_*.png`).

## 3. Технологии и workflow
- **React 18 + Vite**, single-page (роутинг через state, без router-либы).
- Хранилище: **localStorage** через `src/storage.js`, ключи с префиксом `pump_`.
- Telegram SDK: `telegram-web-app.js` в `index.html`, обвязка в `src/tg.js`.
- Графики/иконки/тело — **самописный SVG**. Внешние chart-либы НЕ используем (npm в песочнице заблокирован; CDN-либы — только в виджет-инструменте, не в приложении).

### Правила работы с файлами (ВАЖНО)
- Write/Edit **режут кириллицу** на длинных файлах (NUL-байты) → файлы пишем через **bash heredoc** (`cat > file << 'EOF'`) или python; мелкие правки — python/sed.
- После правки: **баланс скобок** (python `{} () []`) + проверка JSX через `@babel/parser` (`plugins:['jsx']`), для чистых `.js` — `node --check`.
- Логику движков тестируем **node ESM-тестом** (с шимом `localStorage` где нужно). Полную сборку в песочнице не гоняем (rollup/esbuild — win32-бинарь).

## 4. Структура (src/)

### Движки (чистый JS)
- `engine/exercises.js` — `GROUPS`(7), `GROUP_META`(label/short/color), `levelGE`, **`EXERCISES` (135)**: `{id,name,group,equip:'none'|'dumbbell'|'gym',compound,level,muscles,technique[],mistakes[],tip}`, `techExtras(ex)`. Счёт: грудь 20, спина 21, ноги 27, плечи 17, бицепс 14, трицепс 16, пресс 20.
- `engine/sessionGenerator.js` — `generateSession({groups,goal,level,equip,favorites,varied})`, `pickAlternative`, `shapeExercise`; **суперсеты/статистика:** `buildBlocks(exercises,ss)` (группирует в блоки), `blockSlots(block)` (чередование A/B по раундам), `restSeconds(rest)`, `sessionStats(session)` → `{sets,mins,exs}`. `allowedEquip`: зал=`['dumbbell','gym']`, гантели=`['none','dumbbell']`, без=`['none']` (предпочтительный последним).
- `engine/nutrition.js` — `calcNutrition` (Mifflin-St Jeor) → `{bmr,tdee,kcal,protein,fat,carbs}`.
- `engine/loads.js` — `estimateLoad(ex,profile)` → «≈ X кг» / «кг/рука» / «вес тела».
- `engine/progress.js` — `e1rm`, `exSeries`, `personalRecord`, `allRecords`, `weeklyVolume`, `suggestNext` (+2.5кг при верхе диапазона, иначе +1 повтор), `lastSummary`, `isPR`.
- `engine/recovery.js` — карта восстановления: `recoveryMap()`, `coverageMap(days,target)`, `freshFocus(limit)`, `statusOf`, `presetFreshness(groups)`.
- `engine/achievements.js` — 9 медалей (Регулярность/Объём/Сила), `achievementsSummary()`.
- `engine/offapi.js` — Open Food Facts: `searchOFF(query)` (онлайн-поиск еды), `productByBarcode(code)` (продукт по штрихкоду). Fetch с таймаутом, защита от ошибок. CORS-публичный, работает в задеплоенном приложении (в песочнице сеть закрыта).
- `engine/foodscore.js` — `dayScore(eaten,{kcal,protein})` → `{score 0-100, grade A-E, color, tips[]}` (добор белка/попадание в ккал/доля жиров).

### Данные
- `data/programs.js` — `PROGRAMS`(6), `programsForEquip`, `recommendProgram`, `buildWeek`, `activeOrRecommended`.
- `data/foods.js` — **174 продукта СНГ** (10 категорий), `searchFoods`, `macrosFor`.

### storage.js (ключи `pump_*`)
profile `pump_profile_v3`, progress, water, measures, food, fav, favex, rating, logs, freq, dishes, kcalgoal, active, fast, routines, photos.
Методы: профиль (get/set/clear/`clearAll`), progress(`workouts[]`), дневник еды (+`updateFood`, `frequentFoods`, `foodWeek`), избранное (еда+упр), рейтинг, логи(`addLog/getExLogs/getLastSets`), блюда, цель ккал, активная программа, голодание(`getFast/setFast`), **свои тренировки**(`getRoutines/addRoutine/removeRoutine`, хранят и `ss`), **фото**(`getPhotos/addPhoto/removePhoto`, ≤12). `todayKey`, `calcStreak`, `calcWeekStreak`.

### Компоненты
- `App.jsx` — корень: profile/route/loading/catGroup. Нижняя таб-навигация (`NavIcon`) `[home,workout,catalog,progress,more]`; `startGroup`→тренировка на группу; вторичные экраны (profile/nutrition/water/measures/timer/calculators); Loader; Telegram BackButton.
- `Home.jsx` — карта восстановления как фон-сцена (`.bodystage`, body фоном + стеклянные чипы) → герокарта старта → календарь → сводка дня → плитки → совет дня. Тап по мышце → `MuscleSheet`.
- `BodyHeatmap.jsx` — крупная карта тела (`muscle_front/back.png`), переключатель Спереди/Сзади, заливка по `load`; зоны в `bodyZones.js`.
- `MuscleSheet.jsx` — шит по мышце: статус + картинка + «Тренировка на эту группу» / «Упражнения».
- `muscleArt.js` — `MUSCLE_ART` (группа→кроп) + `MUSCLE_HL` (зоны маркера).
- **`MuscleThumb.jsx` (v1.8)** — общий: `Hl` (рендер колец-прицелов) + `MuscleThumb` (мини-картинка мышцы с маркером). Используется в каталоге и тренировках.
- `WorkoutBuilder.jsx` — пресеты, «Собери за меня» (`autoPick` по свежести), ручной выбор групп (с маркерами), генерация сессии. Сессия: **статистика** (≈мин·упр·подходы), чипы групп, схема, гид/другой вариант/замена, **«Сохранить как свою»**, «Мои тренировки», карточки упражнений с **мини-картинкой мышцы+маркером+номером**, **связывание в суперсет** (`toggleSS`, одно упражнение — максимум в один суперсет).
- `GuidedWorkout.jsx` — пошаговый: разминка→работа→финиш. Работает **блоками** (`buildBlocks`/`blockSlots`): суперсет = чередование A→B без отдыха, отдых перед новым раундом; баннер суперсета; вес/повторы запоминаются на каждое упражнение (`inputsRef`); разминочные подходы (флаг `warm`); вспышка PR; вехи на финише; шеринг.
- `Catalog.jsx` — **поиск** по всем упражнениям (имя/мышцы/группа), сетка 7 групп с маркерами; внутри группы — **фильтр по инвентарю** + разбивка **Базовые/Изолирующие**, строки с мини-картинкой мышцы; деталь: герой+маркер, **«Смотреть видео техники»** (`tg.openLink` → поиск YouTube), техника/ошибки/темп-дыхание-разминка.
- `Progress.jsx` — дашборд (вкладки Обзор/Графики/Медали/История): герой со статами+недельная серия+шеринг, медали, карта покрытия 30 дней, рекорды, график веса, тоннаж, оценка программы, история по упражнению.
- `Nutrition.jsx` — калории 4.0: кольцо БЖУ (`NutritionRing`), баланс съедено−сожжено, цели по приёмам (30/35/25/10%), мини-вода + таймер голодания (16:8/18:6/14:10), вкладки (Поиск/Частые/Избранное/Мои блюда/Вручную) + **онлайн-поиск Open Food Facts**, конструктор блюд, правка порции, **×N степпер**, **score качества** (бейдж→шит), **приоритет белка** (г/кг), **повтор вчера/приёма**, **штрихкод** (скан камерой Telegram + ручной ввод → OFF), **недельная аналитика**.
- `More.jsx` — шапка профиля + секции (duotone `MenuIcon`): Тело и питание, Инструменты, Система.
- `tg.js` — `initTelegram`, `tgBackButton`, `haptic`, `tgUserName`, **`openLink`** (внешние ссылки/видео), `shareText`.
- Прочее: `CalendarStrip`, `WeekSchedule`, `Water`, `Measures` (прогресс-фото До/Сейчас), `Timer`, `Calculators`, `Onboarding` (визард + прогноз ±кг/мес), `Profile`, `Loader`, `SparkChart`, `Icon`.

### Шиты (v1.8)
Все всплывающие плашки (`.sheet`) открываются **в верхней трети** (`padding: 11vh`, `align-items:flex-start`), чтобы кнопка действия всегда была видна. Полноэкранный просмотр фото — по центру (`.sheet-center`).
