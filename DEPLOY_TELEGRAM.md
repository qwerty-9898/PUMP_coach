# Деплой PUMP на Vercel + подключение к Telegram

## 1. Выложить на Vercel
Вариант А (проще, через сайт):
1. Залей папку `fit-coach` в репозиторий на GitHub.
2. Зайди на vercel.com → Add New → Project → импортируй репозиторий.
3. Vercel сам определит Vite. Настройки уже заданы в `vercel.json`:
   - Build Command: `npm run build`
   - Output: `dist`
4. Жми Deploy. Получишь ссылку вида `https://pump-xxx.vercel.app`.

Вариант Б (через консоль):
```
npm i -g vercel
cd fit-coach
vercel        # первый раз — логин и пара вопросов, жми Enter
vercel --prod # боевой деплой, выдаст HTTPS-ссылку
```

## 2. Привязать к боту (@BotFather)
Способ 1 — кнопка-меню рядом с полем ввода:
1. Открой @BotFather → /mybots → выбери своего бота.
2. Bot Settings → Menu Button → Edit menu button URL.
3. Вставь ссылку Vercel. Текст кнопки: `Открыть PUMP`.

Способ 2 — отдельный Mini App со своей ссылкой t.me/твойбот/app:
1. @BotFather → /newapp → выбери бота.
2. Заполни название, описание, картинку (640x360).
3. В поле Web App URL вставь ссылку Vercel.
4. Получишь прямую ссылку на приложение для шеринга.

## 3. Проверить
Открой бота в Telegram на телефоне → жми кнопку → должно открыться приложение
на весь экран с нативной шапкой Telegram.

## Обновления
Каждый новый деплой на Vercel автоматически обновляет приложение —
у бота ничего менять не надо, ссылка та же.
