# FitCoach — ИИ-тренер (MVP v0.1)

Персональный план тренировок под параметры человека + расчёт калорий под цель.
PWA на React + Vite. Ставится на телефон как приложение, работает офлайн (план считается на устройстве).

## Что уже работает (v0.1)
- Онбординг-анкета: пол, возраст, вес, рост, цель, уровень, место (дом/гантели/зал), дни в неделю.
- Движок подбора плана: сплит под число дней и уровень (новичку — фуллбади/верх-низ для частоты), упражнения под доступный инвентарь, подходы/повторы/отдых под цель.
- Расчёт калорий и БЖУ (формула Миффлина-Сан Жеора + активность + поправка под цель).
- Подсказки по технике для каждого упражнения (поводырь для новичка — наш главный отличительный признак).
- Сохранение профиля в браузере (localStorage).

## Как запустить (на своём компе)
Нужен Node.js 18+.

```bash
cd fit-coach
npm install        # поставит React и Vite
npm run dev        # откроется на http://localhost:5173
```

Чтобы протестить с телефона в одной Wi-Fi сети — Vite покажет адрес вида `http://192.168.x.x:5173`, открой его на телефоне.

Сборка продакшн-версии:
```bash
npm run build      # положит готовое в папку dist/
npm run preview    # локальный просмотр собранной версии
```

## Структура
```
fit-coach/
├─ index.html
├─ package.json
├─ vite.config.js
├─ public/manifest.json        # PWA-манифест
└─ src/
   ├─ main.jsx                 # точка входа
   ├─ App.jsx                  # роутинг между анкетой и планом
   ├─ styles.css               # тёмная мобильная тема
   ├─ components/
   │  ├─ Onboarding.jsx        # анкета
   │  └─ PlanView.jsx          # экран плана + калории
   └─ engine/
      ├─ exercises.js          # база упражнений
      └─ planGenerator.js      # алгоритм сплита + расчёт калорий
```

## Примечание
В песочнице сборка через npm недоступна (нет доступа к реестру пакетов), поэтому
полный `vite build` нужно запустить у себя. Логика движка плана протестирована
отдельно на 4 разных профилях — работает корректно.
